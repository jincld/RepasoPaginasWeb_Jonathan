Hablemos sobre HTML y CSS

Bienvenido a esta sección del sitio, creada para presentar mis respuestas sobre HTML y CSS. Aquí he resuelto preguntas comunes y proporcionado información para demostrar mi aprendizaje sobre estos lenguajes de programación.

Ir a las preguntas

¿Qué es HTML y cuál es su función principal en el desarrollo web?

El Hyper Text Markup Language es un lengaje de programción a base de etiquetas que se utiliza para dar estructura a páginas web.

¿Qué son las etiquetas HTML y cómo se utilizan?

Las etiquetas en HTML son las que dan forma al código, delimitan cada sección del código para que tenga sentido.

Ejemplos de algunas de las etiquetas HTML más comunes

Entre las más comunes están: body, head, header, div, br, script, h1, h2 y link.

¿Qué es CSS y cómo se utiliza para dar estilo a una página web?

El Cascade Style Sheet es un lenguaje que complementa a HTML, con CSS damos una apariencia vistosa a un documento HTML al llamarlo en el proyecto.

¿Cuáles son las propiedades más importantes de CSS?

Dentro de las propiedades más importantes tenemos: background-color, color, font-size, font-family, position, width, height, entre otras.

Explica el modelo de caja (box model) en CSS.

Dentro del box model obtenemos una guía a seguir para dar un estilo adecuando como el espacio a los lados a los elementos. Su estructura es: margin, padding, border y content.

¿Qué es el diseño responsivo y cómo se logra con CSS?

El diseño responsivo asegura que el sitio web se vea bien en todos los tamaños de pantalla al adaptarse, dentro del CSS definimos con "@media only" los tamaños y ajustes a hacer según estos.
